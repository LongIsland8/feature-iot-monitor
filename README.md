IoT Monitor — Система мониторинга датчиков с уведомлениями
Сервис для приёма данных от IoT-датчиков, анализа по правилам и отправки уведомлений.  
Тестовое задание: реализация backend-части системы мониторинга.

 Функционал

- Приём данных от датчиков через вебхук `/webhooks/sensor`
- Валидация входящих данных (Pydantic)
- Анализ событий по правилам из `rules.json`
- Сохранение событий в PostgreSQL
- Отправка алертов в Telegram при нарушении условий
- Асинхронная обработка через Celery + Redis
- REST API для просмотра событий
- Swagger UI для документации

Используемые технологии
FastAPI — современный фреймворк для создания API
PostgreSQL — хранение событий
Celery + Redis — фоновая обработка задач
Pydantic — валидация данных
Docker Compose — оркестрация сервисов
Swagger UI — интерактивная документация
